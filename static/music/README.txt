Place your running BGM files in this folder with exact names:

- run_calm.mp3   (for 150 BPM / easy running)
- run_steady.mp3 (for 165 BPM / steady running)
- run_rush.mp3   (for 180 BPM / fast running)

How it works:
- Select BPM preset in the running card.
- Press Start Running: music auto plays.
- Pause/Reset Running: music auto pauses.
- You can also tap the "播放配乐/暂停配乐" button manually.

Tips:
- Keep each track around 2-5 minutes; loop playback is enabled.
- Prefer 128kbps+ MP3 for better quality.
